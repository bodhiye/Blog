交换两个变量的值

Written by bodhiye with ♥ on February 28, 2019 in 算法笔记 , 112 words

1、借助第三个变量来实现

C = A; A = B; B = C;

2、利用加减法来实现两个变量的交换

A = A + B; B = A – B; A = A – B;

3、用位异或运算来实现，也是效率最高的

原理：利用一个数异或本身等于0和异或运算符合交换律

A = A ^ B; B = A ^ B; A = A ^ B;
